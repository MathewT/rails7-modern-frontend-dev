# rails7-modern-frontend-dev

### Create new Rails 7 app

```bash
docker-compose run --no-deps web rails new . --force --skip-git  --database=postgresql
```

### Build the image

```bash
docker-compose build
```

### Run locally

```bash
docker-compose up
```
